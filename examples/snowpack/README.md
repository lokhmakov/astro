# Astro Demo

## Getting set up

1. Check out Astro at: https://github.com/snowpackjs/astro

   1. Install and build Astro:

   ```shell
   npm install
   npm run build
   ```

   2. Link Astro:

   ```shell
   npm link
   ```

2. In this project link Astro and install other deps:

```shell
npm link astro
npm install
```

3. Run the Astro dev environment.

```shell
npm start
```

4. Build the website. (Not yet working.)

```shell
npm run build
```
